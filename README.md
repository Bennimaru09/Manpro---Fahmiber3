# TRAXX 2.0  
**Autonomous Solar-Powered IoT Tractor**

TRAXX 2.0 adalah proyek pengembangan **traktor otonom berbasis energi surya dan IoT** yang dirancang sebagai solusi mekanisasi pertanian modern menuju konsep **Agro 4.0**. Proyek ini bersifat **akademik dan non-komersial**, dengan pendekatan teknologi industri dan manajemen proyek profesional.

TRAXX dikembangkan untuk menjawab tantangan utama sektor pertanian, seperti:
- Tingginya kebutuhan tenaga kerja fisik  
- Rendahnya adopsi teknologi di pertanian  
- Ketergantungan pada mesin berbahan bakar fosil  
- Menurunnya minat generasi muda terhadap sektor pertanian  

Dengan sistem **autonomous navigation**, **sensor tanah**, dan **IoT connectivity**, TRAXX mampu beroperasi secara mandiri di lahan pertanian dengan dukungan **energi terbarukan**.
